## Reporting a Vulnerability

I take the security of my digital twin and cognitive frameworks seriously.

If you discover a security vulnerability or an integrity issue within this repository (e.g., exposed keys, injection flaws in the interactive modules), please do **not** open a public issue.

### How to report
Please report the issue confidentially by contacting me directly via:
* **Email:** (See GitHub Profile / Email in commits)
* **LinkedIn:** [Michael Lorenz](https://www.linkedin.com/in/michael-lorenz-battery-and-ai)

I will acknowledge your report within 48 hours and provide an estimated timeframe for remediation.

### Scope
This policy applies to the source code, JSON-LD schemas, and cognitive agent configurations hosted in this repository.
